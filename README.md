How to build and run
--------------------

Required: 
* OpenGL 4.1+
* DirectX 11 (for Windows)
* gcc (for OSX)
* cl.exe (for Windows)
* glfw, glew, glm
* GNU utils

OSX:

```bash
$ mkdir dist
$ make
```

Windows:

```bash
> mkdir dist
> make windows
> make gl # opengl version
> make dx # directx version
```
